# Razorpay Website Clone

A front-end clone of the Razorpay website (https://razorpay.com) 
built using HTML, CSS, and JavaScript. Created for learning purposes 
to practice UI layout, styling, and responsive design techniques.

## Features
- Pixel-accurate recreation of Razorpay's homepage layout
- Navigation bar with links and buttons
- Hero section with text and imagery
- Multi-section page layout

## Tech Stack
HTML · CSS · JavaScript
